# Components

## Main Components

- Arduino Uno
- Ultrasonic Sensor HC-SR04
- Infrared Sensor
- Servo Motors
- Motor Driver
- 16x2 I2C LCD Display
- Robot chassis
- Mechanical arm and claw
- Waste compartments

## Component Purpose

The Arduino handles decision-making. Sensors provide environmental input. Motors and servos perform movement and collection. The LCD provides real-time feedback during testing and demonstration.
