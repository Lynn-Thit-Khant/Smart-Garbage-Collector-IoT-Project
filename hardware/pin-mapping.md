# Pin Mapping

The following pin mapping is based on the project source code.

| Pin / Variable | Purpose |
|---|---|
| `motor_IN1` | Motor control input |
| `motor_IN2` | Motor control input |
| `motor_IN3` | Motor control input |
| `motor_IN4` | Motor control input |
| `TrigPin1` | Ultrasonic sensor trigger |
| `EchoPin1` | Ultrasonic sensor echo |
| `garbage_sensorPin` | Waste detection sensor input |
| `back_IR_sensor` | Rear infrared sensor input |
| Servo on pin 10 | Obstacle servo |
| Servo on pin 12 | Claw servo |
| Servo on A2 | Extension servo |

## Notes

Pin naming should be kept consistent between code, hardware wiring, and troubleshooting notes.
