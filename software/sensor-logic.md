# Sensor Logic

## Ultrasonic Sensor

The ultrasonic sensor measures distance by sending a trigger pulse and reading the returned echo pulse.

It is used to:

- Detect nearby obstacles
- Estimate distance to objects
- Support obstacle avoidance decisions

## Infrared Sensor

The infrared sensor detects the presence of an object or waste item.

It is used to:

- Identify possible waste objects
- Trigger collection behaviour
- Support sorting or detection logic

## Practical Considerations

Sensor readings can be affected by object material, angle, lighting, and distance. Testing should include different object positions and surfaces.
