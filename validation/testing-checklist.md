# Testing Checklist

## Sensor Testing

- [ ] Ultrasonic sensor returns distance values
- [ ] Infrared sensor detects nearby objects
- [ ] Sensor values change when objects are moved closer or farther

## Movement Testing

- [ ] Robot moves forward
- [ ] Robot moves backward
- [ ] Robot turns left
- [ ] Robot turns right
- [ ] Brake function stops movement

## Collection Testing

- [ ] Claw opens and closes correctly
- [ ] Arm extension moves down and up
- [ ] Waste item can be collected
- [ ] Waste item can be released into compartment

## Display Testing

- [ ] LCD powers on
- [ ] LCD displays recyclable count
- [ ] LCD displays non-recyclable count
- [ ] Counts update after collection

## Integration Testing

- [ ] Robot detects waste
- [ ] Robot moves toward target object
- [ ] Robot collects item
- [ ] Robot updates LCD count
