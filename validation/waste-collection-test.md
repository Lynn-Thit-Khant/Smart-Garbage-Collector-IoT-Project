# Waste Collection Test

## Objective

Validate that the robot can detect and collect a waste object.

## Test Steps

1. Place a test object within sensor range.
2. Allow the robot to detect the object.
3. Confirm that the robot stops near the object.
4. Observe the arm extension and claw movement.
5. Confirm that the item is collected.

## Expected Result

The robot should detect the object, activate the collection mechanism, and pick up the item.

## Notes

Collection accuracy depends on item size, robot alignment, and servo positioning.
