# Obstacle Detection Test

## Objective

Validate that the robot detects nearby obstacles and responds before collision.

## Test Steps

1. Place an object in front of the robot.
2. Run the robot.
3. Observe distance readings or behaviour.
4. Confirm the robot stops, turns, or avoids the obstacle.

## Expected Result

The robot should respond when an obstacle is detected within the configured distance threshold.

## Notes

Test with different object sizes and surfaces because ultrasonic readings may vary.
