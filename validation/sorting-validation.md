# Sorting Validation

## Objective

Validate that the robot separates recyclable and non-recyclable counts correctly.

## Test Steps

1. Run a recyclable item collection test.
2. Confirm the recyclable counter increases.
3. Run a non-recyclable item collection test.
4. Confirm the non-recyclable counter increases.
5. Check the LCD display after each test.

## Expected Result

The LCD display should show the correct count for each category.

## Notes

If the sorting mechanism is based on predefined control logic, document the assumptions clearly.
