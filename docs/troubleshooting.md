# Troubleshooting

## LCD does not display text

Check:

- I2C wiring
- LCD power
- SDA/SCL connections
- LCD library setup
- Display initialization code

## Servo does not move

Check:

- Servo power supply
- Signal pin connection
- Servo attach pin in code
- Mechanical blockage

## Robot does not move

Check:

- Motor driver wiring
- Motor power supply
- Control pin mapping
- Battery level

## Ultrasonic reading is unstable

Check:

- Trigger and echo pin wiring
- Object distance
- Sensor angle
- Loose jumper wires

## IR sensor does not detect object

Check:

- Sensor power
- Sensor sensitivity
- Object position
- Ambient lighting conditions
