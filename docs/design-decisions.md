# Design Decisions

## Arduino Platform

Arduino was selected because it supports fast prototyping, sensor integration, and servo motor control.

## Ultrasonic Sensor

The ultrasonic sensor was used for distance measurement and obstacle detection.

## Infrared Sensor

The infrared sensor was used for waste/object detection because it can provide simple object presence feedback.

## Servo-Controlled Arm and Claw

Servo motors were used because they provide position-based movement, which is suitable for opening, closing, lifting, and releasing mechanisms.

## LCD Feedback

The LCD display was included to provide live operational feedback during testing and demonstrations.

## Modular Code

The source code was separated into movement, sensing, collection, and display modules to make the system easier to understand and maintain.
